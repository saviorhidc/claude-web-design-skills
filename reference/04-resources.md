## Learning Path

### Beginner
1. Learn Tailwind CSS basics (official docs tutorial)
2. Set up a Next.js project with shadcn/ui
3. Try v0.dev to generate components from descriptions
4. Copy v0 output into your project and customize

### Intermediate
1. Create a custom Tailwind theme with design tokens
2. Add Framer Motion animations to your components
3. Use Claude Code to build a full page from a design spec
4. Practice the SPADE prompting framework

### Advanced
1. Build a complete design system (tokens, components, animations)
2. Create a `.cursorrules` or `CLAUDE.md` with your design preferences
3. Use the hybrid workflow (v0 for exploration → Claude Code for production)
4. Set up reusable animation variants and component patterns
5. Contribute your findings back to the AI frontend community
